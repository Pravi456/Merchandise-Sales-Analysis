# Merchandise Sales Analysis Dashboard

## Project Overview

This project analyzes merchandise sales data to uncover insights related to product performance, customer behavior, and regional trends. The dashboard is designed to support data-driven decision-making through interactive visualizations.

## Objectives

* Understand overall sales trends
* Identify top-performing products and categories
* Analyze customer demographics
* Evaluate regional sales performance
* Study shipping and rating impact on purchases

## Dataset

The dataset contains transaction-level sales data including product details, customer information, location data, shipping charges, and ratings.

## Data Modeling

To improve performance and maintain a clean structure:

* Product and Location dimension tables were created using **Power Query reference queries**
* Only required columns were extracted
* Duplicate records were removed
* A **star schema model** was implemented

This approach ensures:

* Reduced redundancy
* Better performance
* Data consistency
* Efficient filtering

## Tools & Technologies

* Microsoft Excel — data preparation
* Power BI — visualization & dashboard design
* Power Query — data transformation
* DAX — calculations and measures

## Dashboard Features

* Sales overview KPIs
* Product performance analysis
* Customer demographic insights
* Regional sales comparison
* Shipping trend visualization
* Ratings impact analysis

## Key Insights

* Certain product categories drive the majority of revenue
* Sales patterns show seasonal peaks
* Higher-rated products perform better
* Location influences purchasing behavior

## Business Impact

The dashboard helps improve marketing focus, inventory planning, and customer experience strategies.

## Conclusion

This dashboard transforms raw sales data into actionable insights, enabling smarter and faster business decisions.
